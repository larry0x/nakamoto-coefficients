chain token price threshold(1) coefficient(2) bribe(3)

(1) Percent of voting power needed to halt block production. For most chains using traditional BFT consensus (e.g. Tendermint) this number is 33+1%. Some other consensus algorithms require higher percentages (e.g. Ethereum's Gasper, 50+1%)

(2) The minimum number of nodes or validators that can collude and shutdown the network

(3) Value of the staked tokens owned by the smallest set of nodes who can collude to shutdown the network. One can bribe them with this amount to carry out the action
